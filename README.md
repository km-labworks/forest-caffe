# Forest Caffe

カフェをテーマにした Web アプリです。  
TypeScript + Vite を使用して制作しています。

## Features

- カフェメニュー表示
- 時刻・タイムゾーン管理
- 動的UI更新
- TypeScriptによる型安全な実装
- モジュール分割による保守性向上

## Tech Stack

- TypeScript
- Vite
- HTML
- CSS

## Project Structure

```txt
src/
├─ data/
├─ guards/
├─ images/
├─ types/
├─ ui/
├─ utils/
└─ main.ts
```

## Getting Started

### Install

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
